Step 1: Learn HTML (Structure of a Webpage)

HTML is the backbone of all websites. Here‚Äôs a simple HTML page:

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a simple HTML page.</p>
</body>
</html>

Step 2: Learn CSS (Styling Your Page)

CSS makes your page look good. Example:

body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

h1 {
    color: white;
    text-align: center;
}

To link this CSS to your HTML page:

<link rel="stylesheet" href="styles.css">

Step 3: Learn JavaScript (Adding Interactivity)

JavaScript makes your website dynamic. Example:

<button onclick="sayHello()">Click me!</button>
<script>
    function sayHello() {
        alert("Hello, World!");
    }
</script>

Step 4: Learn Python (Backend Development)

Python is great for building web applications. Example:

# Print "Hello, World!" in Python
print("Hello, World!")

Run this by saving it as script.py and executing:

python script.py
